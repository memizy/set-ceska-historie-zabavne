## 📖 O sadě

Tento repozitář obsahuje data ve formátu **OQSE** (Open Question Set Exchange). 

Sada vás provede českou historií z úhlu, který se v běžných učebnicích často neukazuje. Zjistíte, že i slavní králové, vojevůdci a spisovatelé byli jen lidé se svými (často velmi podivnými) vrtochy.
* **Co se dozvíte:** Tajnosti Karla Hynka Máchy, fobie Václava II., skutečné pozadí pražských defenestrací nebo netradiční koníčky následníků trůnu.
* **Typy otázek:** Sada je velmi pestrá a obsahuje kartičky (`flashcard`), různé typy kvízů (`mcq-single`, `mcq-multi`, `true-false`), doplňovačky (`short-answer`), přiřazování (`match-pairs`), řazení (`sort-items`) a podrobné vysvětlující poznámky (`note`).
